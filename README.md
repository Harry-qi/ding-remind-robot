# 提醒小助手（钉钉机器人）

## 使用方法

1. 增加任务名称、开始时间以及结束时间
用户在钉钉群发送如下示例并@机器人

``` javascript
{
	type:'add',
	data:[
		{
			name: '测试',
			endTime: '2021-02-01 12:00:00',
		}
	]
}
```

2. 修改任务名称、任务开始以及过期时间
用户在钉钉群发送如下示例并@机器人

``` javascript
{
	type:'update',
	data:[
		{
			name:'测试',
			endTime: '2021-02-01 12:00:00',
		}
	]
}
```

3. 删除提醒任务
用户在钉钉群发送如下示例并@机器人

``` javascript
{
	type:'delete',
	data:['测试','测试2']
}
```

4. 查看当前提醒任务列表
用户在钉钉群发送'list'并@机器人

5. 查看帮助
用户在钉钉群发送'help'并@机器人
